グラデーション

グラデーションは線形・円形・角度・方向・色数まで設定することができ、基本的なグラデーションは問題なく作ることができます。
また同時に背景画像や背景色を指定することもできます。 

#grad1 {
	background-image: -webkit-gradient(
						linear, left top, left bottom,
						from(#333), to(#ccc));
	background-image: -moz-linear-gradient(
						#333, #ccc);
	background-image: linear-gradient(
						#333, #ccc);
}

#grad2 {
	background-image: -webkit-gradient(
						linear, left top, right bottom, 
						from(#ffcc00),
						color-stop(0.50, #ff6600),
						color-stop(0.50, #ff3300),
						to(#ff6600));
	background-image: -moz-linear-gradient(
						-45deg,
						#ffcc00 0%,
						#ff6600 50%,
						#ff3300 50%,
						#ff6600);
	background-image: linear-gradient(
						-45deg,
						#ffcc00 0%,
						#ff6600 50%,
						#ff3300 50%,
						#ff6600);
}

#grad3 {
	background-image: -webkit-gradient(radial,
						center center, 0,
						center center, 50,
						from(#9cf), to(#369));
	background-image: -moz-radial-gradient(
						110px 50px,
						circle closest-side,
						#9cf 0%, #369 100%);
	background-image: radial-gradient(
						110px 50px,
						circle closest-side,
						#9cf 0%, #369 100%);
}

ボックスシャドウ

ボックスシャドウはボックスモデルの外側・内側に影を作ることができ、X軸・Y軸・ぼかし具合・色を設定することができます。
また、カンマ（,）で区切ることによって複数の影を作れます。 

#bShadow1 {
	-webkit-box-shadow: 0 0 6px #666;
	-moz-box-shadow: 0 0 6px #666;
	box-shadow: 0 0 6px #666;
}

#bShadow2 {
	-webkit-box-shadow: inset 0 0 6px #666;
	-moz-box-shadow: inset 0 0 6px #666;
	box-shadow: inset 0 0 6px #666;
}

#bShadow3 {
	-webkit-box-shadow: 0 0 6px #f00,
						inset 0 0 6px #00f;
	-moz-box-shadow: 0 0 6px #f00,
					inset 0 0 6px #00f;
	box-shadow: 0 0 6px #f00,
				inset 0 0 6px #00f;
}

テキストシャドウ

ボックスシャドウはテキストの外側に影を作ることができ、X軸・Y軸・ぼかし具合・色を設定することができます。
また、カンマ（,）で区切ることによって複数の影を作れます。

テキスト
シャドウ
#tShadow1 {
	text-shadow: 1px 1px 3px #666;
}
テキスト
シャドウ
#tShadow2 {
	text-shadow: 0 0 6px #666;
	color: #fff;
}
テキスト
シャドウ
#tShadow3 {
	text-shadow: -1px -1px 2px #f00,
				1px 1px 2px #00f;
}
テキスト
シャドウ
#tShadow4 {
	text-shadow: -40px -10px 0 #f00,
				40px 10px 10px #00f;
}

	border-radius: 20px;

新しい色の指定方法と透過

CSS3では色の指定方法に16進数だけではなくRGBを使うことができます。
また、同時に色の透過もできるようになっています。

opacityは要素の透過
#opacity {
	background-color: #f00;
	opacity: 0.5;
}
rgbaによる透過は色の透過
#rgba1 {
	background-color: rgba(255, 0, 0, 0.5);
}
透過が不要な場合は rgb()
#rgba2 {
	background-color: rgb(255, 0, 0);
}
文字にも使えます
#rgba3 {
	background-color: rgb(255, 0, 0);
	color: rgba(255, 255, 255, 0.5);
}

拡大・縮小

CSS3ではtransformプロパティを使うことで要素の拡大・縮小ができます。 

#scale1 {
	background: url(bg3.jpg) no-repeat;
	-webkit-transform: scaleX(0.5);
	-moz-transform: scaleX(0.5);
	transform: scaleX(0.5);
}

#scale2 {
	background: url(bg3.jpg) no-repeat;
	-webkit-transform: scaleY(0.5);
	-moz-transform: scaleY(0.5);
	transform: scaleY(0.5);
}

#scale3 {
	background: url(bg3.jpg) no-repeat;
	-webkit-transform: scale(1.5, 1.5);
	-moz-transform: scale(1.5, 1.5);
	transform: scale(1.5, 1.5);
}

アニメーション（transition）

transitionは:hoverなどと組み合わせ、ユーザーのアクションの反応してアニメーションを行います。
※以下のサンプルはマウスオーバーでアニメーションします。 

#transition1 {
	height: 100px;
	background-color: #ccc;
	-webkit-transition-property: height, background-color;
	-webkit-transition-duration: 2s;
	-webkit-timing-function: linear;
	-webkit-transition-delay: 0;
	-moz-transition-property: height, background-color;
	-moz-transition-duration: 2s;
	-moz-timing-function: linear;
	-moz-transition-delay: 0;
	transition-property: height, background-color;
	transition-duration: 2s;
	timing-function: linear;
	transition-delay: 0;
}

#transition1:hover {
	height: 0;
	background-color: #f00;
}

#transition2 {
	height: 100px;
	background-color: #ccc;
	-webkit-transition-property: -webkit-transform, background-color;
	-webkit-transition-duration: 2s;
	-webkit-timing-function: ease;
	-webkit-transition-delay: 0;
	-moz-transition-property: -moz-transform, background-color;
	-moz-transition-duration: 2s;
	-moz-timing-function: ease;
	-moz-transition-delay: 0;
	transition-property: transform, background-color;
	transition-duration: 2s;
	timing-function: ease;
	transition-delay: 0;
}

#transition2:hover {
	background-color: #f00;
	-webkit-transform: rotate(180deg);
	-moz-transform: rotate(180deg);
	transform: rotate(180deg);
}

	-moz-animation-duration: 10s;
	animation-name: resize, bg;
	animation-timing-function: linear;
	animation-iteration-count: infinite;
	animation-duration: 10s;
}
